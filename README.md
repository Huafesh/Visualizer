# 🎧 Trap Nation Visualizer - Estilo Real

Un clon web interactivo del icónico visualizador de música de Trap Nation. Dibuja un círculo pulsante con barras de frecuencia dinámicas y partículas flotantes que reaccionan a los bajos de la música.

---

## 🚀 Características Principales

* Efecto de pulsación ('bounce') de logotipo sincronizado con las bajas frecuencias.
* Espectro de líneas radiales distribuidas equitativamente simulando el diseño oficial de Trap Nation.
* Generador de partículas de fondo que se mueven a distintas velocidades según el ritmo.

---

## 🛠️ Tecnologías Utilizadas

* HTML5 Canvas
* Web Audio API (AnalyserNode)
* Vanilla JavaScript

---

### Ejecución

1. Levanta un servidor local en el directorio:
   ```bash
   python -m http.server 8000
   ```
2. Abre tu navegador en `http://localhost:8000`, sube un archivo de audio y visualiza el espectro.
